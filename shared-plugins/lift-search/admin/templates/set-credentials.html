About Lift and Amazon Cloud Search

  • To use Lift you will need an Amazon Web Services account. Visit Amazon AWS.
  • It’s a good idea to be familiar with both the Amazon CloudSearch and the Lift Documentation before installing this plug-in.
  • Amazon CloudSearch is not free. The monthly charge begins around $75. Please take time to review the pricing guide before installing.
